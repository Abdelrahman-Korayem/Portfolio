# 🚀 Abdelrahman Korayem – Portfolio

Welcome to my personal portfolio! This website is built to showcase my journey, projects, and skills as a Front-End Developer. It features a modern, responsive design with smooth animations and a clean user interface.

## 🌟 Overview

This project is a centralized hub for my professional identity. It highlights my technical expertise, displays my latest projects, and provides an easy way for clients and recruiters to get in touch with me. The portfolio is designed to be highly interactive and accessible across all devices.

## 🛠 Tech Stack

- **Framework:** Next.js (React)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** React Icons & Lucide React
- **Deployment:** Vercel

## 📂 Key Features

- **Responsive & Modern UI:** Seamless experience on Mobile, Tablet, and Desktop.
- **Dynamic Projects Showcase:** A clean layout to display web development projects.
- **Interactive Contact Form:** Fully functional form to receive messages directly to my email.
- **Performance Optimized:** Fast loading times out of the box thanks to Next.js.
- **Beautiful Animations:** Engaging hover effects and transitions for a better user experience.

## ▶️ Getting Started

To run this project locally, simply follow these steps:

```bash
# Clone the repository (if you haven't already)
git clone https://github.com/Abdelrahman-Korayem/Portfolio.git

# Navigate into the project directory
cd Portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
